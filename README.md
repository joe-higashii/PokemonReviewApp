# PokemonReviewApp

A simple C# Web API project built while following a tutorial by Teddy Smith on YouTube. This project is part of my learning journey to understand how to build and deploy Web APIs using .NET.

## Features

- RESTful API endpoints for managing Pokémon reviews.
- Basic CRUD operations (Create, Read, Update, Delete).
- JSON-based data handling.
- Built with .NET 6 and C#.

## Getting Started

### Prerequisites

- .NET SDK 6.0 or later installed on your machine.
- Visual Studio Code or any preferred code editor.

### Installation

1. Clone the repository:
``
git clone https://github.com/joe-higashii/PokemonReviewApp.git
cd PokemonReviewApp
``
2. Restore dependencies:
``
dotnet restore
``
3. Build the project:
``
dotnet build
``
4. Run the application:
``
dotnet run
``

### Project Structure

```
    PokemonReviewApp/
    ├── Controllers/ # API controllers for handling HTTP requests.
    ├── Properties/ # Project properties and metadata.
    ├── appsettings.json # Application configuration settings.
    ├── Program.cs # Main entry point of the application.
    ├── PokemonReviewApp.csproj # Project file defining dependencies and settings.
    └── README.md # Documentation (this file).
```
